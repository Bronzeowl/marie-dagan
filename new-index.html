My Background

My journey into Web Development started in high school. I took a HTML class because I was curious as to how websites were created. Although I did not pursue coding, the skills that I learned remained with me into my adult life. I completed a Bachelor's in Chemistry, but a desire for a greater challenge brought back my passion for coding. This eventually led me to attend the Houston Coding Bootcamp and make the change official to become a Web Developer. I am excited to continue along this path to use my new skills to create clean and elegant code.
